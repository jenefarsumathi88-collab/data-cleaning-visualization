# Data Cleaning & Visualization Project

## About
This project involves cleaning and visualizing the Titanic dataset.

## Dataset
- Titanic Dataset (891 rows, 12 columns)

## Libraries Used
- Pandas
- Matplotlib
- Seaborn

## Steps Done
1. Loaded the dataset
2. Handled missing values
3. Removed duplicates
4. Created 5 visualizations

## Visualizations
1. Survival Count
2. Age Distribution
3. Survival by Gender
4. Fare vs Age
5. Correlation Heatmap
